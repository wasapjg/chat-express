## How to use
Download the project [or clone the repo](https://github.com/wasapjg/chat-express.git):

```sh
git clone https://github.com/wasapjg/chat-express.git
cd chat-express
```

Install it and run:

```sh
npm install
create the .env file using .env-template
npm start

```
